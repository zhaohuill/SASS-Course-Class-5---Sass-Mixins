Mixins, The Functions of SASS:_

+-In SASS Mixis works like Functions In Javascript. Imagine the Syntax of a Function in Javascript but now change the Declaration "function" for an "@mixin" before the name of the same and the Parameters of the Function with the Syntax of the Variables of SASS starting their names with the Sign "$" and the Option of setting a Default Value in case the LAST PARAMETER is not specified writing "$lastParameter:defaultValue"; After that the Operation is the Same, and they are used Declaring them inside the Brackets "{...}" of Element Styles Typing This:_

@include mixinName($parameter1, $parameter2:200px, $parameter3, $parameter4);

+-You will see an Applied Example in the File "mixin.scss" (The Style Files of the Mixins do not require to take an Underscore "_" before their Name) of this Repository, they are imported in the Main Styles File "styles.scss" in the same way as the others.
